IEEE 1815 - Distributed Network Protocol (DNP3)

Abstract: The DNP3 protocol structure, functions, and interoperable application options (subset levels) are specified. The simplest application level is intended for low-cost distribution feeder devices, and the most complex for full-featured systems.

—IEEE 1815
